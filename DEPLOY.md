# Deploying Supportly to Render

## One-time setup

### 1. Push to GitHub
```bash
cd "C:\Users\jayme\Desktop\Others\apps\my-website"
git init
git add .
git commit -m "initial commit"
# Create a repo on github.com, then:
git remote add origin https://github.com/YOUR_USERNAME/supportly.git
git push -u origin main
```

### 2. Deploy on Render
1. Go to https://render.com and sign in (free tier is fine)
2. Click **New → Web Service**
3. Connect your GitHub repo
4. Render will detect `render.yaml` automatically — click **Apply**
5. The service will build and deploy. First deploy takes ~2 minutes.
6. Your live URL will be `https://supportly-XXXX.onrender.com`

### 3. Verify it works
- Open the live URL — you should see the landing page
- Sign up two accounts in separate incognito windows
- Test: text match → chat → end session
- Test: video match → prejoin → camera check → join → call

## Environment variables (auto-set by render.yaml)
| Variable | Value |
|----------|-------|
| `NODE_ENV` | `production` |
| `SESSION_SECRET` | Auto-generated by Render |
| `DB_PATH` | `/var/data/supportly.db` |

The `/var/data` persistent disk is created automatically. Your SQLite DB survives deploys and restarts.

## Beta testing protocol

### Phase 1 — Text only (do this first)
Test checklist with 2-3 trusted testers:
- [ ] Signup + login + logout
- [ ] Dashboard visible, queue join works
- [ ] Two users match for text session
- [ ] Messages send and receive in real time
- [ ] End session (both sides get redirected)
- [ ] Partner ends session (other side sees "Your partner has left")
- [ ] Report flow (reason selected, submit works)
- [ ] Refresh mid-session stays in session
- [ ] Close tab (partner sees session ended)

UX questions to ask testers:
- Did you know how to leave at any point?
- Did the session feel private?
- Was anything confusing or broken?

### Phase 2 — Video (after text passes)
- [ ] Video match → both reach prejoin
- [ ] Camera allowed on both → preview works
- [ ] One user joins, other sees "waiting for partner"
- [ ] Both join → live call starts
- [ ] Mic toggle mutes/unmutes
- [ ] Camera toggle shows 👤 placeholder
- [ ] End Call button ends for both
- [ ] Partner ends → you see "Your partner ended the call"
- [ ] Leave waiting room before both join → match cancelled cleanly
- [ ] Camera denied → permission error card shows, Join disabled
